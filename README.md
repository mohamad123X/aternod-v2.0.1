# 🎮 Aternod Bot v2.0.1

بوت ديسكورد مبرمج بلغة بايثون للتحكم في تشغيل سيرفرات **Aternos** (ماين كرافت) تلقائياً من خلال أوامر الديسكورد باستخدام تقنية الأتمتة `Playwright`.

---

## ✨ المميزات
* **تشغيل السيرفر بسهولة:** استخدم الأمر `!start` في الديسكورد، وسيقوم البوت بتسجيل الدخول لحسابك في أترنوس والضغط على زر التشغيل نيابة عنك.
* **أوامر مائلة (Slash Commands):** يدعم أوامر ديسكورد المائلة مثل `/setup_aternos` لسهولة الاستخدام والتفاعل.
* **متصفح مخفي (Headless):** يعمل البوت في الخلفية بدون واجهة رسومية، مما يجعله جاهزاً للاستضافة على الخوادم السحابية مثل Railway و Heroku وغيرها.

---

## 🛠️ المتطلبات الأساسية
تأكد من توفر البرامج التالية قبل تشغيل البوت:
* [Python 3.11](https://www.python.org/downloads/) أو أحدث.
* مكتبة `discord.py` للتواصل مع الديسكورد.
* مكتبة `playwright` للتحكم بالمتصفح.
